Missan Group Corporate Website

This professional website was developed for Missan Group, a real construction company, as part of a client-based project experience. Our team collaborated in a structured, professional setting—mirroring real-world workflows, meetings, and client feedback cycles and presentations.

Built with Angular, the site showcases the company's construction services and portfolio. The project involved creating a responsive layout, smooth navigation, dynamic routing, and auto-sliding hero visuals—all while applying service-based architecture and modular coding practices.

Technologies used include Angular Routing, Lazy Loading, Animations, ViewChild DOM access, and CLI deployment to GitHub Pages.

Personal Finance Tracking Application

CashDaddy is a personal finance tracking application built with Java and Gradle. It helps users manage transactions, track expenses, and set budget goals with ease. Users can add, update, and filter transactions by date or category, providing clear insights into their financial activities.

The app features intuitive graphs that visually track savings and expenses, making financial progress easy to understand. With a clean, user-friendly interface, CashDaddy helps users organize their finances and make informed decisions.

This project was developed collaboratively by our team, integrating database CRUD operations.

Web Guys Online Auction

Web Guys Online Auction is an intuitive platform designed to make online auctions simple and efficient. Users can easily sign up, log in, and manage their accounts. Sellers can list items for auction, set starting prices, and choose when the auction will end. Buyers can explore items, place bids, and compete with others in real-time.

This platform includes an automated email notification system that informs the highest bidder when an auction ends. The winner can then securely complete the payment using PayPal.
