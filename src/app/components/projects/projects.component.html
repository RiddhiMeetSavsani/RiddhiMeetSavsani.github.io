Personal Finance Tracking Application

CashDaddy is a personal finance tracking application built with Java and Gradle. It helps users manage transactions, track expenses, and set budget goals with ease. Users can add, update, and filter transactions by date or category, providing clear insights into their financial activities.

The app features intuitive graphs that visually track savings and expenses, making financial progress easy to understand. With a clean, user-friendly interface, CashDaddy helps users organize their finances and make informed decisions.

Check out the GitHub Repository for details.

Web Guys Online Auction

Web Guys Online Auction is a user-friendly platform designed to make online auctions simple and efficient. Users can easily sign up, log in, and manage their accounts. Sellers can list items for auction, set starting prices, and choose when the auction will end. Buyers can explore items, place bids, and compete with others in real-time.

The platform includes an automated email notification system that informs the highest bidder when an auction ends. The winner can then securely complete the payment using PayPal.

Check out the project on the GitHub repository.
